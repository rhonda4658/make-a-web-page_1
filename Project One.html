Lesson One The Basics of the Web and HTML

The major pieces of programming are you, your computer browser, the internet, and servers. There are several computer languages, but the purpose of this course is to learn HTML and CSS. HTML is Hypertext Markup Language. HTML is made up of text content, markup, reference to other documents, as well as links and videos. HTML uses tags to create the webpages. There are too many tags to remember them all. An easy way to reasearch the tags is to use Google.com The tag used to create the link to Google is an attribute tag. It is necessary to list the entire web address in order for the link to be created. Some other very popular tags are b to indicate bold text, em to indicate italics, p to create a new paragraph div to create the boxes, and img to add an image to a website. To write code, programmers use text editors. For this project I am using Sublime.

Lession Two Creating a Structured Document

HTML elements are inline or block. The block elemets create the box design you see on a website if you view the Developer Tools. One of the most common block elements is the div tag. HTML has a treelike structure. It is similar to an outline. The deeper you go into the structure, the more detail is present.

Lession Three Adding CSS Style and HTML Structure

CSS (Cascading Style Sheets) is one way style is added to a webpage. CSS allows style to be applied easier and it avoids REPETITION. There is less code so less chance for errors to be made. In CSS code elements begin and end with "/*", whereas in HTML they begin and end with "< >". ltalics and normal are the values for font-style in CSS. To bold a font in CSS, font-weight is used. Headings are used to index the contents of a webpage. The lower the number, the larger the font size on the page. Class styles are used when only adding style to certain headings. T There are 3 ways to add CSS Styling to a webpage:
  • Write CSS in the head of your HTML. This is good for small projects such as the ones used in Scratchpad
  • Link your HTML to a separate CSS file.
  • Write your style inline with your HTML. This is not recommended because it required a large amount of repeated code.
  • With the box model there are 4 basic parts of the box. They are
    1. content
    2. padding
    3. border
    4. margin
The element width is the found by adding the left border, left padding, content, right padding, and right border together. Box-sizing is a way to create this style. The size can be set as pixels or as a percentage. If it is set as a percentage, the user experience can be better because it adjusts to the size of the screen.

Lesson Four Final notes

One important thing to keep in mind is changing the visual element only changes how it is displayed. It does not change the element itself.

When creating a website there are some general guidelines:

  1. Get the structure right. Visually boxify it.
  2. Look for natural boxes of distinct elements.
  3. Add tags, headings, etc.
  4. Look for repeated styles and semantic elements.
  5. Size the boxes
  6. Apply Styles from biggest to smallest.
  7. And lastly, fix things.
